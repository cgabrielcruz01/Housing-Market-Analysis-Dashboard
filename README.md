# Housing-Market-Analysis-Dashboard

Overview

This project is an interactive Power BI dashboard designed to analyze housing market trends across Canada. Using over 44,000 housing listings, the dashboard provides insights into property prices, housing characteristics, regional market performance, and key real estate metrics. The project demonstrates the use of data cleaning, transformation, modeling, KPI development, and visualization techniques to support data-driven decision-making in the housing market.

Dataset

Source: Canada Housing Dataset (Kaggle)
Dataset Link: https://www.kaggle.com/datasets/yuliiabulana/canada-housing

The dataset includes property prices, property type, number of bedrooms and bathrooms, square footage, location data, property amenities and features.

Project Objectives

Analyze housing prices across Canadian provinces
Compare regional housing market performance
Evaluate property characteristics and amenities
Identify trends in property values
Develop meaningful KPIs for housing market analysis
Create interactive visualizations for stakeholders and decision-makers

Tools used:
Power BI
Power Query
DAX
Data Modeling

Data Preparation

The dataset was cleaned and transformed using Power Query by:
Handling missing and inconsistent values
Standardizing data formats
Creating calculated fields and measures
Preparing data for visualization and KPI reporting
Optimizing the data model for performance
Key Performance Indicators (KPIs)

The dashboard includes several important housing market metrics:
Average Property Price
Median Property Price
Average Square Footage
Average Price per Square Foot
Total Listings
Average Bedrooms per Property
Average Bathrooms per Property

Price Analysis:
Average property prices by province
Median property prices by province
Price distribution analysis
Regional housing market comparisons

Property Characteristics:
Property type analysis
Bedroom and bathroom distributions
Square footage comparisons
Amenity and feature breakdowns

Geographic Analysis:
Province-level market comparisons
Regional performance insights
Interactive geographic filtering

Key Insights:
Housing prices vary significantly across Canadian provinces.
Larger properties generally command higher market values.
Price per square foot differs considerably by region.
Property characteristics such as bedrooms, bathrooms, and square footage have a strong relationship with listing prices.
Regional comparisons reveal varying levels of market affordability across Canada.
